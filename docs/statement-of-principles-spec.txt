rel-statement-of-principles is a simple, open format for indicating content codes of practice, which is embeddable in HTML or XHTML, Atom, RSS, and arbitrary XML.

Abstract
--------

By adding rel="statement-of-principles" to a hyperlink, a page indicates that the destination of that hyperlink is a description of the code of practice the content on the page was created under.

<a href="http://www.mediastandardstrust.org/code-of-practice" rel="statement-of-principles">some code of practice name goes here</a>
