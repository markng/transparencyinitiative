rel-cop is a simple, open format for indicating content codes of practice, which is embedable in HTML or XHTML, Atom, RSS, and arbitrary XML.

* currently unsure whether cop is a desirable abbreviation (or whether there should be an abbreviation at all)

Abstract
--------

By adding rel="cop" to a hyperlink, a page indicates that the destination of that hyperlink is a description of the code of practice the content on the page was created under.

<a href="http://www.mediastandardstrust.org/code-of-practice" rel="cop">some code of practice name goes here</a>
